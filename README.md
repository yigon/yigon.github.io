# Yigon Kim homepage

This is a lightweight Jekyll site for GitHub Pages.

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

Then open the local URL shown by Jekyll, usually <http://127.0.0.1:4000/>.

## Deploy to GitHub Pages

1. Create a repository named `ykim28.github.io` on GitHub.
2. Push this directory to the repository.
3. In GitHub, open **Settings → Pages** and set the source to the main branch/root directory.
4. Edit `_config.yml` if the GitHub username or site URL is different.

## Notes

- Replace placeholder links in `index.md` with your exact Instagram, blog, ADS, and arXiv profile URLs.
- Replace the placeholder dark-matter-halo card in `research.md` if you want to add the missing image.
